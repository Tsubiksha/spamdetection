# 📧 Email Spam Detection using Machine Learning

A Machine Learning project that classifies emails/messages as **Spam** or **Not Spam (Ham)** using **Natural Language Processing (NLP)** techniques and the **Multinomial Naive Bayes** algorithm.

---

## 🚀 Project Overview

Email spam is one of the most common cybersecurity and communication challenges. This project builds a spam detection system that automatically identifies unwanted or fraudulent messages using machine learning.

The system performs:

* Text preprocessing and cleaning
* Stopword removal
* TF-IDF feature extraction
* Spam classification using Naive Bayes
* Prediction on custom user messages

---

## 🎯 Problem Statement

Manually filtering spam emails is inefficient and time-consuming. The goal of this project is to develop an intelligent system capable of automatically classifying incoming messages as:

* **Spam** – Unwanted, promotional, or fraudulent messages
* **Ham** – Legitimate messages

---

## 📂 Dataset

Dataset: SMS Spam Collection Dataset

### Features

| Column | Description         |
| ------ | ------------------- |
| v1     | Label (spam or ham) |
| v2     | Message text        |

### Label Encoding

| Original Label | Encoded Value |
| -------------- | ------------- |
| ham            | 0             |
| spam           | 1             |

## 🔄 Project Workflow

### 1. Data Loading

* Load spam dataset using Pandas
* Select relevant columns

### 2. Data Preprocessing

* Convert text to lowercase
* Remove URLs
* Remove special characters and numbers
* Remove stopwords

### 3. Feature Engineering

* Convert text into numerical vectors using TF-IDF

### 4. Train-Test Split

* 80% Training Data
* 20% Testing Data

### 5. Model Training

* Train Multinomial Naive Bayes classifier

### 6. Model Evaluation

* Accuracy Score
* Precision
* Recall
* F1-Score

### 7. Prediction

* Classify new messages as Spam or Not Spam

---

## 📊 Machine Learning Model

### Multinomial Naive Bayes

Why Naive Bayes?

* Fast and efficient
* Performs well on text classification problems
* Suitable for high-dimensional sparse data generated by TF-IDF
